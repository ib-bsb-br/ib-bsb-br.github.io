Introduction

\n "\n "

This solution leverages a-Shell, SSH, and embedded dash scripts to automate tasks on a remote Debian server. It focuses on efficiency, security, and user feedback through notifications.

\n "\n "

Prerequisites

\n "\n "
    \n "
  • An iPhone with a-Shell installed.
    • \n "
  • A Debian server with SSH access enabled.
    • \n "
  • Basic familiarity with SSH key management.
    • \n "
  • A thermal printer configured on the Debian server.
    • \n "
\n "\n "

Debian Server Setup

\n "\n "

Ensure thermal.sh Functionality

\n "\n "

Since we’ll be embedding the thermal.sh scripts within the a-Shell script, you don’t need to have thermal.sh files on the server. However, ensure that your server has:

\n "\n "
    \n "
  • The lp command available (usually part of the CUPS system).
    • \n "
  • The thermal printer correctly configured and accessible via the lp command.
    • \n "
  • Necessary utilities like iconv installed.
    • \n "
\n "\n "

You can install iconv if it’s not present:

\n "\n "
sudo apt-get update\n    
        
        
        "sudo apt-get install libc-bin\n    
        
        
        "
\n "\n "

SSH Configuration

\n "\n "

Set up SSH key-based authentication to enable passwordless SSH connections from your iPhone to the Debian server.

\n "\n "

Generating SSH Keys on a-Shell

\n "\n "
    \n "
  1. Open a-Shell on your iPhone.
    2. \n "
  2. \n "

    Generate an SSH key pair:

    \n "\n " 
    ssh-keygen -t rsa -b 2048\n    
        
        
        "
    \n "
    3. \n "
  3. Accept the default file location and enter a passphrase if desired (leave empty for no passphrase).
    4. \n "
\n "\n "

Copying Your Public Key to the Debian Server

\n "\n "
    \n "
  1. \n "

    Copy your public key to the server:

    \n "\n " 
    ssh-copy-id yourusername@your_server_ip\n    
        
        
        "
    \n "\n "

    Replace yourusername with your actual username on the Debian server and your_server_ip with the server’s IP address or hostname.

    \n "
    2. \n "
  2. \n "

    Test the SSH connection:

    \n "\n " 
    ssh yourusername@your_server_ip\n    
        
        
        "
    \n "\n "

    You should be able to connect without entering a password.

    \n "
    3. \n "
\n "\n "

Create the a-Shell Script

\n "\n "

Create the remote_control.sh script on your iPhone to pass the text directly to the embedded thermal.sh scripts on the server.

\n "\n "

Steps to Create remote_control.sh

\n "\n "
    \n "
  1. Open a-Shell on your iPhone.
    2. \n "
  2. \n "

    Run the following command to create the script:

    \n "\n " 
    cat > ~/.shortcuts/remote_control.sh << 'EOL'\n    
        
        
        "#!/bin/dash\n    
        
        
        "\n    
        
        
        "# Configuration (Update with your actual credentials)\n    
        
        
        "SSH_USER="yourusername"\n    
        
        
        "SSH_HOST="your_server_ip"\n    
        
        
        "\n    
        
        
        "# Function to display notification in a-Shell\n    
        
        
        "show_notification() {\n    
        
        
        "    message="$1"\n    
        
        
        "    echo "$message" >&2  # Print to stderr for visibility in a-Shell and Shortcuts\n    
        
        
        "}\n    
        
        
        "\n    
        
        
        "# Function to run command with error handling\n    
        
        
        "run_command() {\n    
        
        
        "    command="$1"\n    
        
        
        "    error_message="$2"\n    
        
        
        "    if ! eval "$command"; then\n    
        
        
        "        show_notification "Error: $error_message"\n    
        
        
        "        exit 1\n    
        
        
        "    fi\n    
        
        
        "}\n    
        
        
        "\n    
        
        
        "# Main execution\n    
        
        
        "TEXT_CONTENT="$1"\n    
        
        
        "\n    
        
        
        "# Check if text content is provided\n    
        
        
        "if [ -z "$TEXT_CONTENT" ]; then\n    
        
        
        "    show_notification "Error: No text content provided."\n    
        
        
        "    exit 1\n    
        
        
        "fi\n    
        
        
        "\n    
        
        
        "# SSH command with error handling and passing the text directly to thermal.sh\n    
        
        
        "SSH_COMMAND="ssh -o BatchMode=yes -o ConnectTimeout=5 $SSH_USER@$SSH_HOST"\n    
        
        
        "\n    
        
        
        "# Embedded Normal Thermal Script\n    
        
        
        "NORMAL_THERMAL_SCRIPT='#!/bin/dash\n    
        
        
        "\n    
        
        
        "# Check for input\n    
        
        
        "if [ -z "$1" ]; then\n    
        
        
        "  echo "Error: No text input provided."\n    
        
        
        "  exit 1\n    
        
        
        "fi\n    
        
        
        "\n    
        
        
        "# Escape % to prevent printf interpretation\n    
        
        
        "escaped_content=$(echo "$1" | sed "s/%/%%/g")\n    
        
        
        "\n    
        
        
        "# Convert the encoding while suppressing specific iconv errors\n    
        
        
        "converted_text=$(echo "$escaped_content" | iconv -f UTF-8 -t CP850//TRANSLIT//IGNORE 2>/dev/null)\n    
        
        
        "\n    
        
        
        "# Print the formatted text to the thermal printer\n    
        
        
        "if ! printf "\033@%s\n\n\n\033i" "$converted_text" | lp -d thermal -o raw; then\n    
        
        
        "  echo "Failed to print. Please check the printer and its configuration."\n    
        
        
        "  exit 1\n    
        
        
        "fi\n    
        
        
        "\n    
        
        
        "echo "Success: Text printed successfully."\n    
        
        
        "'\n    
        
        
        "\n    
        
        
        "# Create and execute the normal thermal script on the server\n    
        
        
        "run_command "echo \"$NORMAL_THERMAL_SCRIPT\" | $SSH_COMMAND 'cat > /tmp/normal_thermal.sh && chmod +x /tmp/normal_thermal.sh'" "Failed to transfer normal thermal script."\n    
        
        
        "run_command "echo \"$TEXT_CONTENT\" | $SSH_COMMAND '/tmp/normal_thermal.sh \"\$(cat -)\"'" "Failed to execute normal thermal script."\n    
        
        
        "\n    
        
        
        "# Optionally, you can comment out the reverse thermal script execution if not needed\n    
        
        
        "# Embedded Reverse White Thermal Script\n    
        
        
        "REVERSE_THERMAL_SCRIPT='#!/bin/dash\n    
        
        
        "\n    
        
        
        "# Check for input\n    
        
        
        "if [ -z "$1" ]; then\n    
        
        
        "  echo "Error: No text input provided."\n    
        
        
        "  exit 1\n    
        
        
        "fi\n    
        
        
        "\n    
        
        
        "# Escape % to prevent printf interpretation\n    
        
        
        "escaped_content=$(echo "$1" | sed "s/%/%%/g")\n    
        
        
        "\n    
        
        
        "# Convert the encoding while suppressing specific iconv errors\n    
        
        
        "converted_text=$(echo "$escaped_content" | iconv -f UTF-8 -t CP850//TRANSLIT//IGNORE 2>/dev/null)\n    
        
        
        "\n    
        
        
        "# Print the formatted text to the thermal printer with reverse white printing\n    
        
        
        "if ! printf "\032[\001\000\000\000\000@\002\260\004\000\032T\001\000\000\000\000\030\000\004\000%s\000\032]\000\032O\000\033i" "$converted_text" | lp -d thermal -o raw; then\n    
        
        
        "  echo "Failed to print. Please check the printer and its configuration."\n    
        
        
        "  exit 1\n    
        
        
        "fi\n    
        
        
        "\n    
        
        
        "echo "Success: Reverse white text printed successfully."\n    
        
        
        "'\n    
        
        
        "\n    
        
        
        "# Uncomment the following lines to execute the reverse thermal script\n    
        
        
        "# run_command "echo \"$REVERSE_THERMAL_SCRIPT\" | $SSH_COMMAND 'cat > /tmp/reverse_thermal.sh && chmod +x /tmp/reverse_thermal.sh'" "Failed to transfer reverse thermal script."\n    
        
        
        "# run_command "echo \"$TEXT_CONTENT\" | $SSH_COMMAND '/tmp/reverse_thermal.sh \"\$(cat -)\"'" "Failed to execute reverse thermal script."\n    
        
        
        "\n    
        
        
        "show_notification "Success: Text printed successfully."\n    
        
        
        "EOL\n    
        
        
        "
    \n "
    3. \n "
  3. Replace yourusername and your_server_ip with your actual SSH username and server IP address or hostname.
    4. \n "
  4. \n "

    Make the script executable:

    \n "\n " 
    chmod +x ~/.shortcuts/remote_control.sh\n    
        
        
        "
    \n "
    5. \n "
\n "\n "

Script Explanation

\n "\n "
    \n "
  • SSH Configuration: The script uses ssh with options -o BatchMode=yes and -o ConnectTimeout=5 for non-interactive and timeout settings.
    • \n "
  • Notifications: The show_notification function sends messages back to the Shortcuts app.
    • \n "
  • Error Handling: The run_command function checks if each command executes successfully.
    • \n "
  • Embedded Scripts: Both the normal and reverse thermal scripts are embedded as strings within the remote_control.sh script.
    • \n "
  • Execution Flow:\n "
      \n "
    • The embedded script is sent to the server and saved as a temporary file in /tmp/.
      • \n "
    • The script is made executable.
      • \n "
    • The text content is passed to the script for printing.
      • \n "
    • Cleanup of temporary scripts can be added if desired.
      • \n "
    \n "
    • \n "

iOS Shortcut Setup

\n "\n "
    \n "
  1. Open the Shortcuts app on your iPhone.
    2. \n "
  2. Create a new shortcut and give it a descriptive name (e.g., “Print to Thermal Printer”).
    3. \n "
  3. Add a Text action:\n "
      \n "
    • Leave it blank to prompt for input when the shortcut runs.
      • \n "
    • Or, enter the text you wish to print.
      • \n "
    \n "
    4. \n "
  4. Add a Run Script Over SSH action (note that this action is available in the Shortcuts app):\n "
      \n "
    • Host: Enter localhost (since we’re running the script locally on a-Shell).
      • \n "
    • User: Leave blank.
      • \n "
    • Password: Leave blank.
      • \n "
    • \n "

      Script: Enter the following command:

      \n "\n " 
      shortcuts://run-shortcut?name=Run%20a-Shell%20Script&input={Text}\n    
        
        
        "
      \n "\n "

      (Note: Since a-Shell does not support running scripts over SSH directly from Shortcuts, we use the URL scheme to run the script via a-Shell.)

      \n "
      • \n "
    \n "
    5. \n "
  5. Add a URL action:\n "
      \n "
    • \n "

      Set the URL to:

      \n "\n " 
      a-shell://shortcut?text={Text}&command=~/.shortcuts/remote_control.sh%20"{Text}"\n    
        
        
        "
      \n "\n "
        \n "
      • Replace {Text} with the magic variable from the previous Text action.
        • \n "
      \n "
      • \n "
    \n "
    6. \n "
  6. Add an Open URLs action:\n "
      \n "
    • Use the URL from the previous step.
      • \n "
    • This will open a-Shell and run the remote_control.sh script with the provided text.
      • \n "
    \n "
    7. \n "
  7. Add a Show Notification action (optional):\n "
      \n "
    • This will display the output from the script after it completes.
      • \n "
    \n "
    8. \n "
\n "\n "
