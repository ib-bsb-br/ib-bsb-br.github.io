## 1. INSTALLATION

### 1.1 Primary Requirements
- **PowerShell 7+** (`pwsh`)
- **Optional extractors:**
  - **7z** for `.7z` archives
  - **unrar** for `.rar` archives
- **tar** (usually available on Linux/macOS) for `.tar`, `.gz`, `.bz2` archives

Validate prerequisites:
```bash
pwsh --version
7z --help   # optional
unrar       # optional
tar --help
```

### 1.2 Script File Reference
The provided script is typically named `f4p.ps1`, so you would run:
```bash
pwsh ./f4p.ps1 [OPTIONS] [PATHS...]
```

---

## 2. BASIC USAGE

Run the script as follows:
```bash
pwsh ./f4p.ps1 [OPTIONS] /some/path /another/path
```
- The script recursively scans directories and extracts recognized archives:
  - `.zip`, `.rar`, `.7z`, `.tar`, `.gz`, `.bz2`
  - Office Open XML / OpenDocument formats: `.docx`, `.xlsx`, `.pptx`, `.odt`, `.ods`, `.odp`

*(Legacy formats such as `.doc`, `.xls`, `.ppt` are not treated as archives.)*

---

## 3. FLAGS & OPTIONS

| Flag/Option         | Type / Default       | Short | Description                                                     |
|---------------------|----------------------|-------|-----------------------------------------------------------------|
| `-Extension`       | string[] / empty       | `-e`  | Restricts processing to specific extensions (archives & Office docs). |
| `-IncludeHidden`  | switch / False         | n/a   | Includes hidden/dot files and directories.                     |
| `-IgnoreGitignore`| switch / False         | n/a   | Ignores `.gitignore` rules in directories.                      |
| `-Ignore`          | string[] / empty       | n/a   | Excludes files matching glob patterns (e.g., `*.log`).            |
| `-Output`          | file path / None       | `-o`  | Writes output to the specified file instead of stdout.          |
| `-Xml`             | switch / False         | n/a   | Outputs content in an XML-like structure.                       |
| `-TemplateFile`   | file path / None       | `-t`  | Uses token replacement for custom formatting (`{{ path }}`, `{{ content }}`, `{{ index }}`).   |
| `-MaxDepth`       | int / 5                | `-d`  | Limits recursion depth for nested archives (default: 5).          |

## 4. FILE & ENCODING LOGIC

1. **Multiple Encoding Attempts:**
   - Tries `utf-8` first, then `latin-1`.
   - If both fail, the file's content is omitted (with a warning logged).

2. **Archive & Office Document Extraction:**
   - `.rar` extraction uses `unrar` when available, and `.7z` extraction uses `7z` when available.
   - Office documents (OOXML/ODF) are processed as `.zip` archives.
   - Extraction occurs in temporary folders and unsupported extractors emit warnings.

---

## 5. IGNORING LOGIC

1. **.gitignore Handling:**
   - The script reads `.gitignore` lines and applies wildcard matching (`-like`) against file names and paths.
   - This is a pragmatic matcher and not a full `gitwildmatch` implementation.

2. **Hidden Files:**
   - Hidden items are skipped by default (unless `-IncludeHidden` is used).

3. **Additional Patterns:**
   - Use `-Ignore` to exclude files (e.g., `-Ignore "*.log"` or `-Ignore "*_backup.*"`).

---

## 6. OUTPUT FORMATS

1. **Plain Text (Default):**
   - Prints each file’s path, followed by a separator and the file content.

2. **XML-like (`-Xml`):**
   - Wraps the content within `<section>...</section>` elements.

3. **Template File (`-t`/`-TemplateFile`):**
   - Applies token replacement in a text template to format each file's content.

## 7. EXAMPLES

1. **Restrict to `.ps1` & `.md` and ignore `*.log`:**
   ```bash
   pwsh ./f4p.ps1 -Extension .ps1 -Extension .md -Ignore "*.log" /path/to/process
   ```

2. **Process hidden files, disable `.gitignore`, and output to a file:**
   ```bash
   pwsh ./f4p.ps1 -IncludeHidden -IgnoreGitignore -Output out.txt /some/path
   ```

3. **Output as XML and limit recursion to 3 levels:**
   ```bash
   pwsh ./f4p.ps1 -Xml -MaxDepth 3 /path/to/archives
   ```

4. **Use a template file:**
   ```bash
   pwsh ./f4p.ps1 -TemplateFile custom_template.txt /path/to/files
   ```

5. **Process multiple paths:**
   ```bash
   pwsh ./f4p.ps1 /first/path /second/path
   ```

## 8. TEMPLATE EXAMPLES

When invoking:
```bash
pwsh ./f4p.ps1 -TemplateFile my_template.txt [PATHS...]
```
the template supports direct token replacement for:
- `{{ path }}`: file path
- `{{ content }}`: full file content
- `{{ index }}`: running index

### Valid Token-Based Examples

1. **Minimal text wrapper** (`minimal_example.txt`)
```text
File #{{ index }}: {{ path }}
----------------------------
{{ content }}
----------------------------
```

2. **Markdown snippet** (`markdown_snippet.txt`)
```text
### File #{{ index }}: {{ path }}

~~~text
{{ content }}
~~~
```

3. **JSON-style envelope** (`json_envelope.txt`)
```text
{
  "index": "{{ index }}",
  "path": "{{ path }}",
  "content": "{{ content }}"
}
```

4. **HTML wrapper** (`html_output.txt`)
```text
<html>
  <body>
    <h2>{{ path }}</h2>
    <pre>{{ content }}</pre>
  </body>
</html>
```

> Note: this implementation does **not** execute Jinja logic (`{% ... %}` / filters / loops).
> It only performs literal token replacement.

### D2. Control-flow algorithm

Observed high-level algorithm from the source:

1. Initialize strict-mode and error behavior.
2. Normalize extension filters.
3. Open output writer if requested.
4. Load template if requested.
5. Emit XML root opener if XML mode is active and no template is used.
6. For each input path:

   * verify existence;
   * resolve the path;
   * dispatch to `Invoke-ProcessPath`.
7. `Invoke-ProcessPath`:

   * reject over-depth;
   * if file, call `Invoke-FileProcessing`;
   * if directory, call `Invoke-DirectoryProcessing`;
   * otherwise warn.
8. `Invoke-FileProcessing`:

   * if archive-like, extract to a temp directory and recurse into it;
   * else read text and write formatted content.
9. `Invoke-DirectoryProcessing`:

   * merge inherited and local `.gitignore` rules;
   * enumerate children in sorted order;
   * skip hidden items unless allowed;
   * skip `.gitignore`-matched items unless disabled;
   * recurse into directories;
   * apply file-level ignore and extension filters;
   * process eligible files.
10. On exit:

* emit XML closer if needed;
* dispose output writer.

### D3. Notable implementation behaviors

* Output is streamed rather than accumulated into one large in-memory string.
* Text reading uses explicit .NET stream handling rather than `Get-Content`.
* Archive handling is separated by format family.
* The script reads file contents fully into memory once it begins reading a file.

#!/usr/bin/env pwsh

[CmdletBinding()]
param(
    [Parameter(Mandatory, Position = 0, ValueFromRemainingArguments)]
    [string[]]$Paths,

    [Alias('e')]
    [string[]]$Extension = @(),

    [switch]$IncludeHidden,
    [switch]$IgnoreGitIgnore,

    [string[]]$Ignore = @(),

    [Alias('o')]
    [string]$Output,

    [switch]$Xml,

    [Alias('t')]
    [string]$TemplateFile,

    [Alias('d')]
    [ValidateRange(0, 100)]
    [int]$MaxDepth = 5
)

Set-StrictMode -Version 3.0
$ErrorActionPreference = 'Stop'

$script:OutputWriter = $null
$script:TemplateText = $null
$script:XmlIndex = 1
$script:OfficeExtensions = @('.docx', '.xlsx', '.pptx', '.odt', '.ods', '.odp')
$script:ArchiveExtensions = @('.zip', '.rar', '.7z', '.tar', '.gz', '.bz2')
$script:AllArchiveLikeExtensions = @($script:ArchiveExtensions + $script:OfficeExtensions)
$script:NormalizedExtensions = @(
    $Extension |
        Where-Object { -not [string]::IsNullOrWhiteSpace($_) } |
        ForEach-Object {
            if ($_.StartsWith('.')) { $_.ToLowerInvariant() } else { ".$(($_).ToLowerInvariant())" }
        } |
        Select-Object -Unique
)

function Write-OutputLine {
    param(
        [AllowNull()]
        [string]$Text = ''
    )

    if ($null -ne $script:OutputWriter) {
        $script:OutputWriter.WriteLine($Text)
    } else {
        [Console]::Out.WriteLine($Text)
    }
}

function Test-IsWithinDirectory {
    param(
        [Parameter(Mandatory)][string]$Directory,
        [Parameter(Mandatory)][string]$Target
    )

    $directoryPath = [System.IO.Path]::GetFullPath($Directory)
    $targetPath = [System.IO.Path]::GetFullPath($Target)
    $comparison = if ($IsWindows) {
        [System.StringComparison]::OrdinalIgnoreCase
    } else {
        [System.StringComparison]::Ordinal
    }

    $trimChars = [char[]]@(
        [System.IO.Path]::DirectorySeparatorChar,
        [System.IO.Path]::AltDirectorySeparatorChar
    )
    $normalizedDirectory = $directoryPath.TrimEnd($trimChars)
    $directoryPrefix = $normalizedDirectory + [System.IO.Path]::DirectorySeparatorChar

    return $targetPath.Equals($normalizedDirectory, $comparison) -or
           $targetPath.StartsWith($directoryPrefix, $comparison)
}

function Test-IsHidden {
    param(
        [Parameter(Mandatory)]
        [System.IO.FileSystemInfo]$Item
    )

    return $Item.Name.StartsWith('.') -or
           (($Item.Attributes -band [System.IO.FileAttributes]::Hidden) -eq [System.IO.FileAttributes]::Hidden)
}

function Read-GitIgnoreRules {
    param(
        [Parameter(Mandatory)]
        [string]$Directory
    )

    $gitIgnorePath = Join-Path -Path $Directory -ChildPath '.gitignore'
    if (-not (Test-Path -LiteralPath $gitIgnorePath -PathType Leaf)) {
        return @()
    }

    return @(
        Get-Content -LiteralPath $gitIgnorePath -Encoding utf8 |
            ForEach-Object { $_.Trim() } |
            Where-Object { $_ -and -not $_.StartsWith('#') }
    )
}

function Test-RuleMatch {
    param(
        [Parameter(Mandatory)][string]$Candidate,
        [Parameter(Mandatory)][string]$Rule,
        [switch]$IsDirectory
    )

    $normalizedCandidate = ($Candidate -replace '\\', '/').TrimStart('./')
    $normalizedRule = ($Rule -replace '\\', '/').Trim()

    if ([string]::IsNullOrWhiteSpace($normalizedCandidate) -or
        [string]::IsNullOrWhiteSpace($normalizedRule)) {
        return $false
    }

    $candidateSet = [System.Collections.Generic.List[string]]::new()
    $leaf = Split-Path -Path $normalizedCandidate -Leaf

    foreach ($value in @($leaf, $normalizedCandidate)) {
        if (-not [string]::IsNullOrWhiteSpace($value) -and -not $candidateSet.Contains($value)) {
            $candidateSet.Add($value)
        }
    }

    if ($IsDirectory) {
        foreach ($value in @("$leaf/", "$normalizedCandidate/")) {
            if (-not [string]::IsNullOrWhiteSpace($value) -and -not $candidateSet.Contains($value)) {
                $candidateSet.Add($value)
            }
        }
    }

    foreach ($value in $candidateSet) {
        if ($value -like $normalizedRule) {
            return $true
        }
    }

    return $false
}

function Test-ShouldIgnore {
    param(
        [Parameter(Mandatory)][string]$Path,
        [Parameter(Mandatory)][string]$BaseDirectory,
        [Parameter(Mandatory)][string[]]$Rules,
        [switch]$IsDirectory
    )

    if ($Rules.Count -eq 0) {
        return $false
    }

    try {
        $relativePath = [System.IO.Path]::GetRelativePath($BaseDirectory, $Path)
    } catch {
        $relativePath = Split-Path -Path $Path -Leaf
    }

    $candidates = @(
        (Split-Path -Path $Path -Leaf),
        ($relativePath -replace '\\', '/'),
        ('./' + ($relativePath -replace '\\', '/'))
    ) | Where-Object { -not [string]::IsNullOrWhiteSpace($_) } | Select-Object -Unique

    foreach ($rule in $Rules) {
        foreach ($candidate in $candidates) {
            if (Test-RuleMatch -Candidate $candidate -Rule $rule -IsDirectory:$IsDirectory) {
                return $true
            }
        }
    }

    return $false
}

function Test-ExtensionFilter {
    param(
        [Parameter(Mandatory)]
        [string]$FileName
    )

    if ($script:NormalizedExtensions.Count -eq 0) {
        return $true
    }

    $lowerName = $FileName.ToLowerInvariant()
    foreach ($extension in $script:NormalizedExtensions) {
        if ($lowerName.EndsWith($extension)) {
            return $true
        }
    }

    return $false
}

function Get-TemplateContent {
    if ([string]::IsNullOrWhiteSpace($TemplateFile)) {
        return
    }

    if (-not (Test-Path -LiteralPath $TemplateFile -PathType Leaf)) {
        throw "Template file not found: $TemplateFile"
    }

    $script:TemplateText = Get-Content -LiteralPath $TemplateFile -Raw -Encoding utf8
}

function Write-FormattedContent {
    param(
        [Parameter(Mandatory)][string]$Path,
        [Parameter(Mandatory)][string]$Content
    )

    if ($null -ne $script:TemplateText) {
        $rendered = $script:TemplateText.
            Replace('{{ path }}', $Path).
            Replace('{{ content }}', $Content).
            Replace('{{ index }}', [string]$script:XmlIndex)
        Write-OutputLine -Text $rendered
        $script:XmlIndex++
        return
    }

    if ($Xml) {
        $escapedPath = [System.Security.SecurityElement]::Escape($Path)
        $escapedContent = [System.Security.SecurityElement]::Escape($Content)
        Write-OutputLine -Text "<section><source>$escapedPath</source><contents>$escapedContent</contents></section>"
        $script:XmlIndex++
        return
    }

    Write-OutputLine -Text $Path
    Write-OutputLine -Text '--'
    Write-OutputLine -Text $Content
    Write-OutputLine -Text ''
    Write-OutputLine -Text '--'
    $script:XmlIndex++
}

function Read-TextFile {
    param(
        [Parameter(Mandatory)]
        [string]$Path
    )

    $encodings = @(
        [System.Text.UTF8Encoding]::new($false, $true),
        [System.Text.Encoding]::GetEncoding('ISO-8859-1')
    )

    foreach ($encoding in $encodings) {
        $stream = $null
        $reader = $null
        try {
            $stream = [System.IO.File]::OpenRead($Path)
            $reader = [System.IO.StreamReader]::new($stream, $encoding, $false)
            return $reader.ReadToEnd()
        }
        catch [System.Text.DecoderFallbackException] {
            Write-Verbose "Decoding failed for '$Path' using $($encoding.WebName). Trying next encoding."
        }
        catch {
            Write-Warning "File read error $Path: $($_.Exception.Message)"
            return $null
        }
        finally {
            if ($null -ne $reader) {
                $reader.Dispose()
            } elseif ($null -ne $stream) {
                $stream.Dispose()
            }
        }
    }

    Write-Warning "Could not read file $Path with provided encodings."
    return $null
}

function Get-ExternalCommandName {
    param(
        [Parameter(Mandatory)]
        [string[]]$Candidates
    )

    foreach ($candidate in $Candidates) {
        $command = Get-Command -Name $candidate -ErrorAction SilentlyContinue | Select-Object -First 1
        if ($null -ne $command) {
            return $command.Name
        }
    }

    return $null
}

function Expand-ZipArchiveSafe {
    param(
        [Parameter(Mandatory)][string]$FilePath,
        [Parameter(Mandatory)][string]$Destination
    )

    Add-Type -AssemblyName System.IO.Compression.FileSystem
    $archive = $null
    try {
        $archive = [System.IO.Compression.ZipFile]::OpenRead($FilePath)
        foreach ($entry in $archive.Entries) {
            if ([string]::IsNullOrEmpty($entry.FullName)) {
                continue
            }

            $destinationPath = Join-Path -Path $Destination -ChildPath $entry.FullName
            $fullDestinationPath = [System.IO.Path]::GetFullPath($destinationPath)

            if (-not (Test-IsWithinDirectory -Directory $Destination -Target $fullDestinationPath)) {
                throw "Detected path traversal attempt in archive entry '$($entry.FullName)'."
            }

            if ($entry.FullName.EndsWith('/')) {
                [System.IO.Directory]::CreateDirectory($fullDestinationPath) | Out-Null
                continue
            }

            $parent = Split-Path -Path $fullDestinationPath -Parent
            if (-not [string]::IsNullOrWhiteSpace($parent)) {
                [System.IO.Directory]::CreateDirectory($parent) | Out-Null
            }

            $entryStream = $null
            $outputStream = $null
            try {
                $entryStream = $entry.Open()
                $outputStream = [System.IO.File]::Create($fullDestinationPath)
                $entryStream.CopyTo($outputStream)
            }
            finally {
                if ($null -ne $outputStream) { $outputStream.Dispose() }
                if ($null -ne $entryStream) { $entryStream.Dispose() }
            }
        }

        return $true
    }
    catch {
        Write-Warning "ZIP extraction failed for $FilePath: $($_.Exception.Message)"
        return $false
    }
    finally {
        if ($null -ne $archive) {
            $archive.Dispose()
        }
    }
}

function Expand-TarArchiveSafe {
    param(
        [Parameter(Mandatory)][string]$FilePath,
        [Parameter(Mandatory)][string]$Destination
    )

    $tarCommand = Get-ExternalCommandName -Candidates @('tar')
    if ($null -eq $tarCommand) {
        Write-Warning "TAR handling requires 'tar' to be installed and available in PATH."
        return $false
    }

    try {
        $entries = @(& $tarCommand -tf $FilePath 2>$null)
        if ($LASTEXITCODE -ne 0) {
            throw "Unable to list archive contents with tar."
        }

        foreach ($entry in $entries) {
            if ([string]::IsNullOrWhiteSpace($entry)) {
                continue
            }

            if ($entry.StartsWith('/') -or $entry -match '^[A-Za-z]:[\\/]') {
                throw "Detected absolute path in archive entry '$entry'."
            }

            $segments = $entry -split '[\\/]'
            if ($segments -contains '..') {
                throw "Detected path traversal attempt in archive entry '$entry'."
            }

            $targetPath = [System.IO.Path]::GetFullPath((Join-Path -Path $Destination -ChildPath $entry))
            if (-not (Test-IsWithinDirectory -Directory $Destination -Target $targetPath)) {
                throw "Detected path traversal attempt in archive entry '$entry'."
            }
        }

        & $tarCommand -xf $FilePath -C $Destination 2>$null
        if ($LASTEXITCODE -ne 0) {
            throw "tar extraction returned exit code $LASTEXITCODE."
        }

        return $true
    }
    catch {
        Write-Warning "TAR extraction failed for $FilePath: $($_.Exception.Message)"
        return $false
    }
}

function Expand-ArchiveWithSevenZip {
    param(
        [Parameter(Mandatory)][string]$FilePath,
        [Parameter(Mandatory)][string]$Destination,
        [Parameter(Mandatory)][string]$ArchiveKind
    )

    $sevenZip = Get-ExternalCommandName -Candidates @('7zz', '7z')
    if ($null -eq $sevenZip) {
        Write-Warning "$ArchiveKind handling requires '7zz' or '7z' to be installed."
        return $false
    }

    try {
        & $sevenZip x "-o$Destination" -y $FilePath 2>$null
        if ($LASTEXITCODE -ne 0) {
            throw "7-Zip extraction returned exit code $LASTEXITCODE."
        }
        return $true
    }
    catch {
        Write-Warning "$ArchiveKind extraction failed for $FilePath: $($_.Exception.Message)"
        return $false
    }
}

function Invoke-ArchiveExtraction {
    param(
        [Parameter(Mandatory)][string]$Path,
        [Parameter(Mandatory)][string]$Extension,
        [Parameter(Mandatory)][int]$Depth
    )

    $tempDirectory = Join-Path -Path ([System.IO.Path]::GetTempPath()) -ChildPath ([System.IO.Path]::GetRandomFileName())
    [System.IO.Directory]::CreateDirectory($tempDirectory) | Out-Null

    try {
        $success = switch ($Extension) {
            '.zip'  { Expand-ZipArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.docx' { Expand-ZipArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.xlsx' { Expand-ZipArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.pptx' { Expand-ZipArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.odt'  { Expand-ZipArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.ods'  { Expand-ZipArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.odp'  { Expand-ZipArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.tar'  { Expand-TarArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.gz'   { Expand-TarArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.bz2'  { Expand-TarArchiveSafe -FilePath $Path -Destination $tempDirectory }
            '.7z'   { Expand-ArchiveWithSevenZip -FilePath $Path -Destination $tempDirectory -ArchiveKind '7z' }
            '.rar'  { Expand-ArchiveWithSevenZip -FilePath $Path -Destination $tempDirectory -ArchiveKind 'RAR' }
            default { $false }
        }

        if ($success) {
            Invoke-ProcessPath -Path $tempDirectory -Depth ($Depth + 1) -GitIgnoreRules @()
        } else {
            Write-Warning "Extraction failed for $Path"
        }
    }
    finally {
        Remove-Item -LiteralPath $tempDirectory -Recurse -Force -ErrorAction SilentlyContinue
    }
}

function Invoke-FileProcessing {
    param(
        [Parameter(Mandatory)][string]$Path,
        [Parameter(Mandatory)][int]$Depth
    )

    $extension = [System.IO.Path]::GetExtension($Path).ToLowerInvariant()
    if ($script:AllArchiveLikeExtensions -contains $extension) {
        Invoke-ArchiveExtraction -Path $Path -Extension $extension -Depth $Depth
        return
    }

    $content = Read-TextFile -Path $Path
    if ($null -ne $content) {
        Write-FormattedContent -Path $Path -Content $content
    }
}

function Invoke-DirectoryProcessing {
    param(
        [Parameter(Mandatory)][string]$Directory,
        [Parameter(Mandatory)][int]$Depth,
        [string[]]$GitIgnoreRules = @()
    )

    if ($Depth -gt $MaxDepth) {
        Write-Warning "Max recursion depth ($MaxDepth) reached at $Directory."
        return
    }

    $effectiveRules = @($GitIgnoreRules)
    if (-not $IgnoreGitIgnore) {
        $effectiveRules += @(Read-GitIgnoreRules -Directory $Directory)
    }

    $children = @(
        Get-ChildItem -LiteralPath $Directory -Force:$IncludeHidden -ErrorAction Stop |
            Sort-Object -Property Name
    )

    foreach ($child in $children) {
        if (-not $IncludeHidden -and (Test-IsHidden -Item $child)) {
            continue
        }

        if (-not $IgnoreGitIgnore) {
            $isIgnored = Test-ShouldIgnore -Path $child.FullName -BaseDirectory $Directory -Rules $effectiveRules -IsDirectory:$child.PSIsContainer
            if ($isIgnored) {
                continue
            }
        }

        if ($child.PSIsContainer) {
            Invoke-ProcessPath -Path $child.FullName -Depth ($Depth + 1) -GitIgnoreRules $effectiveRules
            continue
        }

        if ($Ignore.Count -gt 0) {
            $skipByPattern = $false
            foreach ($pattern in $Ignore) {
                if ($child.Name -like $pattern) {
                    $skipByPattern = $true
                    break
                }
            }
            if ($skipByPattern) {
                continue
            }
        }

        if (-not (Test-ExtensionFilter -FileName $child.Name)) {
            continue
        }

        Invoke-ProcessPath -Path $child.FullName -Depth ($Depth + 1) -GitIgnoreRules $effectiveRules
    }
}

function Invoke-ProcessPath {
    param(
        [Parameter(Mandatory)][string]$Path,
        [Parameter(Mandatory)][int]$Depth,
        [string[]]$GitIgnoreRules = @()
    )

    if ($Depth -gt $MaxDepth) {
        Write-Warning "Max recursion depth ($MaxDepth) reached at $Path."
        return
    }

    if (Test-Path -LiteralPath $Path -PathType Leaf) {
        Invoke-FileProcessing -Path $Path -Depth $Depth
        return
    }

    if (Test-Path -LiteralPath $Path -PathType Container) {
        Invoke-DirectoryProcessing -Directory $Path -Depth $Depth -GitIgnoreRules $GitIgnoreRules
        return
    }

    Write-Warning "Path not found or unsupported: $Path"
}

try {
    if (-not [string]::IsNullOrWhiteSpace($Output)) {
        $directory = Split-Path -Path $Output -Parent
        if (-not [string]::IsNullOrWhiteSpace($directory) -and -not (Test-Path -LiteralPath $directory -PathType Container)) {
            [System.IO.Directory]::CreateDirectory($directory) | Out-Null
        }

        $script:OutputWriter = [System.IO.StreamWriter]::new(
            $Output,
            $false,
            [System.Text.UTF8Encoding]::new($false)
        )
    }

    Get-TemplateContent

    if ($Xml -and [string]::IsNullOrWhiteSpace($TemplateFile)) {
        Write-OutputLine -Text '<root>'
    }

    foreach ($path in $Paths) {
        if (-not (Test-Path -LiteralPath $path)) {
            throw "Path does not exist: $path"
        }

        $resolvedPath = (Resolve-Path -LiteralPath $path).ProviderPath
        Invoke-ProcessPath -Path $resolvedPath -Depth 0 -GitIgnoreRules @()
    }

    if ($Xml -and [string]::IsNullOrWhiteSpace($TemplateFile)) {
        Write-OutputLine -Text '</root>'
    }
}
finally {
    if ($null -ne $script:OutputWriter) {
        $script:OutputWriter.Dispose()
    }
}
